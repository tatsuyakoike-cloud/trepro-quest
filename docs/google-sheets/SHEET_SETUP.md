# トレプロクエスト スプレッドシート連携セットアップ

マスターデータは以下のスプレッドシートで管理します。

**スプレッドシートURL:**  
https://docs.google.com/spreadsheets/d/12qHhMQB7DsYZauA64slzs3ABaMcJYMPWivMYgWzqESM/edit

## シート構成

| シート名 | 内容 |
|---------|------|
| **ユーザー** | ログインアカウント（メール・パスワード・権限） |
| **メンバー** | 浅井さん・中岫さんのマスター |
| **ミッション** | カリキュラム・合格条件・審査者 |
| **進捗** | 各メンバー×ミッションの進捗データ |
| **称号** | レベルごとの称号 |
| **ダッシュボード** | 全体サマリー（自動更新） |

## 初回セットアップ手順

### 1. Apps Script を設置

1. スプレッドシートを開く
2. **拡張機能** → **Apps Script**
3. `docs/google-sheets/apps-script/Code.gs` の内容を貼り付けて保存

### 2. マスターデータを投入

1. Apps Script エディタで関数 `setupMasterData` を選択
2. **実行** ボタンをクリック（初回は権限承認が必要）
3. スプレッドシートに6つのシートが作成され、データが入ります

### 3. Webアプリとしてデプロイ

1. **デプロイ** → **新しいデプロイ**
2. 種類: **ウェブアプリ**
3. 実行ユーザー: **自分**
4. アクセス: **全員**
5. デプロイして **URLをコピー**

### 4. アプリに URL を設定

`public/config.json` の `syncApiUrl` にデプロイURLを設定:

```json
{
  "syncApiUrl": "https://script.google.com/macros/s/xxxxx/exec"
}
```

GitHub Pages 公開時は `main` に push すれば反映されます。

## データの更新方法

### スプレッドシートで直接編集

- **ミッション** シート: カリキュラム・合格条件を編集 → UIに自動反映（30秒ごとに同期）
- **進捗** シート: ステータス・合否・フィードバック等を編集 → UIに反映
- **ユーザー** シート: アカウント追加・パスワード変更

### UIから編集

- 管理画面またはクエスト画面から編集 → **進捗** シートに書き込み → ダッシュボードも自動更新

## 合格条件の書き方

**ミッション** シートの `pass_criteria` 列に、パイプ `|` 区切りで記載:

```
サービス内容を正確に説明できる|敬語や言葉遣いに大きな問題がない|商談の基本的な流れを再現できる
```

## トラブルシューティング

| 症状 | 対処 |
|------|------|
| ログインできない | ユーザー シートのメール・パスワードを確認 |
| データが古い | ページをリロード（30秒ごとに自動更新） |
| シートがない | `setupMasterData` を再実行 |
| 保存できない | `syncApiUrl` が正しく設定されているか確認 |
