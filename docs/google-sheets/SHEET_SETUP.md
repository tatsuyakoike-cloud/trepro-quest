# トレプロクエスト スプレッドシート連携セットアップ

マスターデータは以下のスプレッドシートで管理します。

**スプレッドシートURL:**  
https://docs.google.com/spreadsheets/d/12qHhMQB7DsYZauA64slzs3ABaMcJYMPWivMYgWzqESM/edit

## シート構成

| シート名 | 内容 |
|---------|------|
| **ユーザー** | ログインアカウント（メール・パスワード・権限） |
| **メンバー** | 浅井さん・中岫さんのマスター |
| **ミッション** | カリキュラム・合格条件・審査者 |
| **進捗** | 各メンバー×ミッションの進捗データ |
| **称号** | レベルごとの称号 |
| **ダッシュボード** | 全体サマリー（自動更新） |

## 初回セットアップ手順

### 1. Apps Script を設置

1. スプレッドシートを開く
2. **拡張機能** → **Apps Script**
3. `docs/google-sheets/apps-script/Code.gs` の内容を貼り付けて保存

### 2. マスターデータを投入

1. Apps Script エディタで関数 `setupMasterData` を選択
2. **実行** ボタンをクリック（初回は権限承認が必要）
3. スプレッドシートに6つのシートが作成され、データが入ります

### 3. Webアプリとしてデプロイ

1. **デプロイ** → **新しいデプロイ**
2. 種類: **ウェブアプリ**
3. 実行ユーザー: **自分**
4. アクセス: **全員**
5. デプロイして **URLをコピー**

### 4. アプリに URL を設定

`public/config.json` の `syncApiUrl` にデプロイURLを設定:

```json
{
  "syncApiUrl": "https://script.google.com/macros/s/xxxxx/exec"
}
```

GitHub Pages 公開時は `main` に push すれば反映されます。

**代替:** GitHub リポジトリの Secrets に `VITE_SYNC_API_URL` を登録しても連携できます（`config.json` が空の場合に使用）。

### 5. 双方向同期トリガーを設置

1. Apps Script エディタで関数 **`installTriggers`** を選択して実行
2. `setupMasterData` 実行時にも自動で設置されます

これにより、**進捗** シートを直接編集したときも **ダッシュボード** と **メンバー** シートの称号が自動更新されます。

## 双方向同期の仕組み

| 方向 | 動作 |
|------|------|
| **UI → シート** | 管理画面・クエスト画面で編集 → **進捗** シートに書き込み → **ダッシュボード**・**メンバー**（称号）を自動再計算 |
| **シート → UI** | **進捗** シートを直接編集 → `onEdit` トリガーでダッシュボード更新 → UIは15秒ごと（または手動同期）で最新データを取得 |

画面上部の **「シート連携」** バッジが緑色なら連携中、**「ローカル」** なら未連携（ブラウザ内のみ）です。

## 連携確認手順

1. `syncApiUrl` を設定してアプリを開く
2. 画面上部が **「シート連携」** になっていることを確認
3. UIでステータスを変更 → スプレッドシートの **進捗**・**ダッシュボード** シートが更新されるか確認
4. スプレッドシートの **進捗** シートを直接編集 → 15秒以内（または更新ボタン）でUIに反映されるか確認

## データの更新方法

### スプレッドシートで直接編集

- **進捗** シート: ステータス・合否・フィードバック等を編集 → ダッシュボードが自動更新 → UIに反映（15秒ごと）
- **ミッション** シート: カリキュラム・合格条件を編集 → UIに反映
- **ユーザー** シート: アカウント追加・パスワード変更

### UIから編集

- 管理画面またはクエスト画面から編集 → **進捗** シートに書き込み → **ダッシュボード**・**メンバー**（称号）も自動更新

## 合格条件の書き方

**ミッション** シートの `pass_criteria` 列に、パイプ `|` 区切りで記載:

```
サービス内容を正確に説明できる|敬語や言葉遣いに大きな問題がない|商談の基本的な流れを再現できる
```

## トラブルシューティング

| 症状 | 対処 |
|------|------|
| ログインできない | ユーザー シートのメール・パスワードを確認 |
| データが古い | 画面上部の更新ボタンをクリック、または15秒待つ |
| 画面上部が「ローカル」 | `syncApiUrl` が未設定。Apps Script をデプロイして URL を設定 |
| シートがない | `setupMasterData` を再実行 |
| 保存できない | `syncApiUrl` が正しく設定されているか確認 |
